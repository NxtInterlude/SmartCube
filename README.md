# SmartCube
<b> V2.0.1 BETA</b> 

### Features
* Particle REST api
* Particle api feed/event support
* MQTT Support
* EEPROM persistent data
* <a href="https://www.smartthings.com/">SmartThings</a> support

### Animations
* Fireworks
* ZPlasma
* Squarral
* Snake by Ben (Cubetube.org)
* Life 3D by Ben (Cubetube.org)
* Puck Dude by Batman (Cubetube.org)

###Dependencies
